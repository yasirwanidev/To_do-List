# 📝 To-Do List Web Application

This is a simple and interactive To-Do List web application built using HTML, CSS, and JavaScript. It allows users to manage their daily tasks efficiently through a clean and user-friendly interface. Users can add tasks, mark them as completed, and delete them when no longer needed.

The application uses browser local storage to save tasks, ensuring that data remains available even after refreshing or reopening the page.

## 🚀 Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Store tasks using local storage

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
